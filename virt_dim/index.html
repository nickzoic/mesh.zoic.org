Virtual Localization for Mesh Network Routing

Nick Moore / Y. Ahmet Sekercioglu / Gregory K. Egan

Abstract:

We present a novel distributed 'virtual localization' method for non-mobile mesh networks, such as sensor meshes, which does not depend on radio ranging or the existence of anchor nodes. We show in simulation that it can be used for establishing efficient and reliable location-based routing information without excessive overheads.
